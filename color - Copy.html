Library

The library represents a place where a peaceful and fresh environment lies. The cool color system brings out the calming nature present in the library. An aggressive color system would contrast the untold message within a library.

Hospital

Hospitals are alarming to all, therefore a bold color scheme is necessary. This bold color scheme brings out urgency, therefore the light shade transform this urgency to a minimal sense. Hospitals are not meant to cause urgency, however when urgency arise they present the treatment to minimize it similar to the bold color to light shade scheme.
