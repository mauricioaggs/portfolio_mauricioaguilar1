Unit 1: HTML

  1. What do you think are the most important vocabulary terms (or key concepts) in this unit? What do you think that they mean?
    • Vocabulary includes such as the tag head, which includes general information about the document, including its title and links to its scripts and style sheets.
    • The body tag element contains all the contents of an HTML document, such as text, hyper links, images, tables, etc.
    • The paragraph element or "p" tag includes information surrounding a topic within the HTML document.
    • All headings like h1 being the most important therefore largest, and h6 being the least important and therefore largest.
    • Another term is list, which organizes all thoughts within the html. Unordered list allow organizations by bullets and ordered list allow organizations by bullets.
    • Doctype tag is the first tag, which allows instruction to the web browser about what version of HTML the page is written in.
    • The Title tag succeeds the doctype tag and allows a name that differentiates it from other work. All these skills are essential in the makeup of this unit and HTML in totality.
    1. What do you think are the most important skills that you learned in this unit of study?
    • The most important skills must be the structure of html, and the ability to make clickable links that allow organization.
    1. In what ways have you gotten better at this kind of work?
    • I have gotten better in all facets such as formatting a document, when to use tags, and the recognition of my mistakes.
    1. In what ways do you think you need to improve?
    • I need to focus when list or paragraph usage fits and I need to check my code throughly.
    1. What problems did you encounter while you were working on this piece? How did you solve them?
    • The first problem was where to place the main tag, but after reviewing my organization, I knew that the main tag didn't have to be in the beginning of the code. It could be at the start of the list of units too.
    • The second problem was how to create a clickable link on the units. I realized I didn't put quotation marks after checking my code, which are essentially to the creation of a clickable link.
    1. What was especially satisfying to you about either the process or the finished product?
    • I always thought creating a web page would be impossible, but after learning the in and outs I feel like I could finally accomplish this.
    1. If you were the teacher, what comments would you make about this piece?
    • At times I may had done unnecessary work although I got the job done. For example putting all eight units instead of the necessary six.
